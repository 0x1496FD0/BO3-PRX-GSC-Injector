# BO3-PRX-GSC-Injector
A Black Ops 3 1.33 prx .gscc injector plugins for GoldHEN working on any jailbreakable firmware
# Installation
  1. Drop the .prx inside __/data/GoldHEN/plugins__
  2. Register the .prx for your BO3 CUSA in __/data/GoldHEN/plugins.ini__
     <img width="1222" height="213" alt="image" src="https://github.com/user-attachments/assets/ac279cd7-ef03-4fb3-b361-c89652df178c" />
  3. Drop your __.gscc__ files inside __/data/BO3 PRX GSC Injector__ and rename them accordingly to follow the format gscc_{__n__} where __n__ is the index of the script (__ex: gscc_0  gscc_1  gscc_2  ...__)
     (yeah i know it suck but the orbis kernel fn to list dir entries fail for some reason)
